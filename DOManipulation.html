DOM Manipulation using jQuery

jQuery is a cross-platform JavaScript library designed to simplify the client-side scripting of HTML. jQuery's syntax make easier to navigate a document, select DOM elements, create animations, handle events, and develop Ajax applications.

1- Hide on MouseOver with callback

				
//Effects: Hiden elements
$("h2#first").mouseover(function(){
  $(this).hide("slow", function(){
   alert("Title hidden");
  });
});

2- Hide on click - Toggle Property

		
//Effects: Hiden elements
$("#btn1").click(function(){
  $("h2#second").toggle("slow");	
});

3- Change on Doble click

		
//Changen Text doble click
$("h2#third").dblclick(function(){
  $(this).html("jQuery HTML");
});

4- Text appended on clic

		
$("h2#fourth").click(function(){
  $(this).append(" using jQuery");
});
